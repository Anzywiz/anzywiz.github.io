06 · Improvement opportunities

Fields available on site, not yet collected

Easy
promotion_end_date
Sale countdown timers appear on some promotional PDPs. Scraping the timer element would populate this field for discounted products.
Easy
material_info (coverage improvement)
Currently at 33.5%. Many brands list fabric composition in the specifications JSON — parsing and promoting it to material_info would raise coverage significantly.
Medium
delivery
Delivery information (e.g. "Express delivery available") appears on product pages for select items. Structured extraction would improve the current 0.4% fill rate.
Medium
size_info (coverage improvement)
Currently at 82.9%. Missing rows are products without size selectors (accessories, etc.). Identifying and tagging these as "one-size" would document coverage gaps explicitly.
